Daily Habits Tracker ✅

Daily Habits Tracker is a simple web app that helps users build consistent routines by tracking everyday habits in a clean, minimal interface. 
Users can add habits, mark them as complete for the current day, and instantly see their overall daily progress.

---

Features

- Add and manage custom daily habits (e.g., Drink Water, Study, Workout).  
- Checkboxes to mark each habit as completed for today.  
- Automatic calculation of today’s progress as a percentage.  
- Visual progress bar that fills based on completion.  
- Data persistence with `localStorage` so habits and status survive page reloads.  
- Responsive, minimal UI suitable for desktop and mobile.

---

Tech Stack

- HTML – Structure and layout of the tracker.  
- CSS – Styling, card layout, colors, progress bar, and responsive design.  
- JavaScript (vanilla) – Habit logic, state management, and localStorage integration.
